## 問題文
Q10.[Forensics] River

JPEGという画像ファイルのフォーマットでは、撮影時の日時、使われたカメラ、位置情報など様々な情報(Exif情報)が付加されることがあるらしい。
この情報から、写真に写っている川の名前を特定して欲しい。

問題ファイル： river.jpg

FLAGの形式は、"cpaw{river_name}"
例：隅田川 → cpaw{sumidagawa}

## 解答
ダウンロードしたjpgの詳細情報から緯度軽度を確認
* 緯度：31° 35' 2.76" N
* 経度：130° 32' 51.726" E

Googlemapから鹿児島県の川であることを確認
